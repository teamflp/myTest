Configuration Git

GitHub : 1- Initialisation de git: git init (Crée le dossier .git) 2- Confoguration: git config --global user.name "nom" et git config --global user.email "email" 3- git remote add origin https://github.com/teamflp/myTest.git 4- git branch (Voir la branche sur laquelle on travail) 5- git add * (Ajouter les fichier) 6- git commit -m "message" (Commiter les fichiers) -7 git status (Voir les fichiers qui ont été modifié) 8- git push origin master (Envoyer les fichiers sur le serveur) git branch -M rf-test git checkout master
